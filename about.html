About this project

This is a simple static website built as a demo for Kerala train timings. It uses client-side JavaScript to filter and present a small sample dataset. To make it production-grade, replace the sample data with a backend API (for example, a REST endpoint that returns JSON), implement authentication for user bookings, and integrate payment APIs if required.

Deploying with GitHub Pages

  1. Create a GitHub repository and push these files.
  2. On GitHub go to Settings → Pages. Choose the branch (`main`) and folder (`/root`) to publish.
  3. After a minute your site will be available at `https://<your-username>.github.io/<repo-name>/`.

Extending

  • Use a real timetable API or CSV import for real data.
  • Add user accounts and persistent bookings.
  • Improve accessibility and localization for Malayalam/English.
